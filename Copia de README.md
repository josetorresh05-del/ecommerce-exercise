# Wilderness Editorial

A high-alpine e-commerce store for durable, sustainable mountain equipment — crafted for explorers who demand performance and environmental responsibility.

## Author

Jose Torres

## Description

Wilderness Editorial is a product card grid and e-commerce page built for the Web & App Technologies course. It features three pages: a Home page with hero, categories, and featured products; a Packs page with a filterable product grid; and a Product Detail page with image gallery, color/size selectors, and accordion specs.

## Pages

- `index.html` — Home page
- `packs.html` — Product listing page with sidebar filters
- `product.html` — Product detail page (Black Hole Pack 25L)

## Tech Stack

- HTML5
- CSS3 (custom properties, CSS Grid, Flexbox)
- Vanilla JavaScript

## Design System

- **Primary:** `#004E66` (Alpine Teal)
- **Secondary:** `#4B5320` (Forest Olive)
- **Tertiary:** `#D2B48C` (Highland Tan)
- **Fonts:** Bebas Neue (headlines), DM Serif Display (editorial), DM Sans (body)
